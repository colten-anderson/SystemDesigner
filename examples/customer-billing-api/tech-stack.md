# Tech Stack — Customer Billing API

- **Language/runtime:** Go 1.24
- **API framework:** gRPC + JSON gateway
- **Data stores:** PostgreSQL 16, Redis 7
- **Queue/eventing:** AWS SNS/SQS
- **Infrastructure:** AWS EKS, RDS, ElastiCache, Terraform
- **Observability:** OpenTelemetry, Prometheus, Grafana, Datadog logs
- **CI/CD:** GitHub Actions + Argo CD
